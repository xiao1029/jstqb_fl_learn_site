# JSTQB Foundation Level 学習サイト(静的HTML一式)

## 内容
- `index.html` ... トップページ
- `chapter1/` 〜 `chapter6/` ... 各章トップページ + 各節ページ(全22節)
- `glossary/` ... 用語集(全章のキーワードを集約)
- `assets/style.css` ... 共通スタイルシート(ticket/exam-objectiveテーマ)

## ローカルで確認する方法
このフォルダ内で以下を実行してください。

    python3 -m http.server 8000

ブラウザで http://localhost:8000/ を開くと閲覧できます。
(サイドバー等のリンクはサイトルート起点の絶対パス `/chapter1/` 形式のため、
 サブディレクトリにデプロイする場合はパスの調整が必要です)

## デプロイ
Vercel / Netlify などの静的ホスティングに、このフォルダをそのままアップロードすれば公開できます
(メダカ管理帳と同じくVercelで運用するのが手軽です)。

## 再生成・編集方法
本文コンテンツは `data_ch123.py` / `data_ch456.py` に章・節・項ごとの
辞書データとして格納されています。文言を直したい場合はこの2ファイルを編集し、

    python3 build.py

を実行すると、22節+6章トップ+トップ+用語集のHTMLがすべて再生成されます。
(HTMLを直接編集するとbuild.py再実行時に上書きされるため、本文修正は
 必ずdata_ch*.pyに対して行ってください)

## 学習目的の認知レベルの色分け
- K1(記憶) = 青系バッジ
- K2(理解) = 緑系バッジ
- K3(適用) = 朱色バッジ(演習が必要な項目の目印)
